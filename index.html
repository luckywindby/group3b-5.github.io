CPT208 Campus Tour · Process Portfolio

Discover campus through routes, landmarks, and practical guidance.

Campus Compass is a campus tour experience designed for new students, visitors, and event participants. It helps users explore landmarks, navigate efficiently, discover useful destinations, and follow practical themed routes such as first-day essentials and orientation-friendly campus guidance.

What makes it different?

Instead of only showing a map, the product turns campus exploration into guided journeys with suggested durations, stop highlights, walking difficulty, and clearer visit priorities.

Core use cases

Help first-time users find must-visit buildings, help visitors tour campus efficiently, and help students move through campus with clearer route suggestions and destination context.

Technical Reflection

How this website was built

The technical framing below is designed to be easy to submit, easy to edit, and easy to deploy.

Implementation approach

  • Built as a single static index.html file with embedded CSS and lightweight JavaScript.
  • No framework or build step is required, which makes it suitable for direct GitHub Pages deployment.
  • The visual style uses gradients, glassmorphism-inspired cards, and responsive grid layouts to improve presentation quality.

Verification and ethics

  • All generated content should be reviewed by the team for correctness and consistency with the actual project scope.
  • AI support is limited to layout generation, text refinement, and prototyping assistance.
  • Project logic, personas, route priorities, and final academic claims remain human-reviewed decisions.
